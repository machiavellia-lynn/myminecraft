
# Minecraft 1.21.1 Mods Setup

⚠️ **Important:** These mods are **only compatible with Minecraft version 1.21.1**.  
They will not work with any other version.

---

## Installation Guide

### 1. Install TLauncher
Download and install TLauncher from the official website:  
[Install TLauncher](https://tlauncher.org/en/)

---

### 2. Add Mods
Copy all files from the `Mods 1.21.1` folder into your Minecraft `mods` directory.

C:\\Users\\yourname\\AppData\\Roaming.minecraft\\mods


---

### 3. (Optional) Install Complementary Shaders
A complementary shader pack is included:  
`ComplementaryUnbound_r5.6.1.zip`

Place this file into your Minecraft `shaderpacks` folder.

C:\\Users\\yourname\\AppData\\Roaming.minecraft\\shaderpacks


---

### 4. Complementary Installer
A **Complementary Installer** is also provided in case you need it later.  
This can help with shader setup or troubleshooting.

---

## Notes
- Ensure you are running **Minecraft 1.21.1** in TLauncher before launching with these mods.
- Settings -> controls -> key binds (to customize your key bind)
- If you encounter issues, double-check that the files are in the correct folders.  
- Shaders are optional but recommended for enhanced visuals.
